# Happy Shisha - Premium Mobile Shisha Experience

A modern, luxury website for mobile shisha services featuring scrollytelling animations and interactive design.

## Features

- Full-screen hero section with animated smoke effects
- Parallax scrolling and smooth section transitions
- Animated counters and fade-in typography
- Mobile-first responsive design
- WhatsApp integration for bookings
- Contact form with validation
- SEO optimized

## Tech Stack

- React 18
- Tailwind CSS
- Framer Motion (animations)
- Vite (build tool)
- Lucide React (icons)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Update WhatsApp number:
Edit `src/components/WhatsAppButton.jsx` and replace the `whatsappNumber` with your actual WhatsApp number.

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

5. Preview production build:
```bash
npm run preview
```

## Customization

### Colors
Edit `tailwind.config.js` to customize the color palette:
- `charcoal`: Background dark color
- `amber`: Accent color (shisha coal glow)
- `smoke`: Secondary gray
- `soft-white`: Text color

### Content
Update text content in individual component files located in `src/components/`.

### Images
Add your images to the `public` folder and reference them in the components.
